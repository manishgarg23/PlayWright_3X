# VS Code Keyboard Shortcuts for JavaScript Development

## Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl + /` | Toggle line comment (`//`) |
| `Shift + Alt + A` | Toggle block comment (`/* */`) |
| `Alt + Up / Down` | Move line up / down |
| `Shift + Alt + Up / Down` | Duplicate line up / down |
| `Ctrl + Shift + K` | Delete current line |
| `Ctrl + Enter` | Insert line below |
| `Ctrl + Shift + Enter` | Insert line above |
| `Ctrl + D` | Select next occurrence of word |
| `Ctrl + Shift + L` | Select all occurrences of word |
| `Alt + Click` | Add multiple cursors |
| `Ctrl + Shift + I` | Format document (Prettier) |
| `Shift + Alt + F` | Format selection |
| `Tab` | Indent line |
| `Shift + Tab` | Outdent line |
| `Ctrl + ]` | Indent line |
| `Ctrl + [` | Outdent line |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl + G` | Go to line number |
| `Ctrl + P` | Quick open file |
| `Ctrl + Shift + O` | Go to symbol in file (functions, variables) |
| `Ctrl + T` | Go to symbol in workspace |
| `F12` | Go to definition |
| `Alt + F12` | Peek definition |
| `Shift + F12` | Show all references |
| `Ctrl + -` | Go back |
| `Ctrl + Shift + -` | Go forward |
| `Ctrl + Home` | Go to top of file |
| `Ctrl + End` | Go to bottom of file |
| `Ctrl + Shift + \` | Jump to matching bracket |

## Search & Replace

| Shortcut | Action |
|----------|--------|
| `Ctrl + F` | Find in file |
| `Ctrl + H` | Find and replace in file |
| `Ctrl + Shift + F` | Search across all files |
| `Ctrl + Shift + H` | Replace across all files |
| `F3` | Find next match |
| `Shift + F3` | Find previous match |
| `Alt + Enter` | Select all find matches |

## IntelliSense & Refactoring

| Shortcut | Action |
|----------|--------|
| `Ctrl + Space` | Trigger IntelliSense / autocomplete |
| `Ctrl + Shift + Space` | Show parameter hints |
| `F2` | Rename symbol (variable/function) |
| `Ctrl + .` | Quick fix / auto import |
| `Shift + Alt + R` | Refactor (extract function, etc.) |
| `Ctrl + K Ctrl + I` | Show hover info for symbol |

| `Shift + F11` | Step out |
